# Handwritten to Digital Text Conversion

## Project Title
Handwritten to Digital Text Conversion

## Short Description
This project aims to convert handwritten text into digital format.

## Features
- Accurate conversion of handwritten text to digital text
- User-friendly interface
- Supports multiple languages

## Tech Stack
- Python
- Flask
- TensorFlow
- OpenCV
- Tesseract OCR

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Vinoth1615/Handwritten-to-digital-text-final.git
   cd Handwritten-to-digital-text-final
   ```
2. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the app/script:
   ```bash
   python app.py
   ```

## Usage Steps
1. Provide an input image of handwritten text.
2. The application will process the image and output the corresponding digital text.

## Project Structure
- `app.py` - Main application script
- `requirements.txt` - List of dependencies
- `static/` - Static files (CSS, images)
- `templates/` - HTML templates

## Notes on Model/OCR Dependencies
- Ensure Tesseract is installed on your machine.
- Specific version requirements may apply for TensorFlow and OpenCV.

## Troubleshooting
- If you encounter errors, check the environment setup and ensure all dependencies are correctly installed.
- Consult the issue tracker for common problems and solutions.
